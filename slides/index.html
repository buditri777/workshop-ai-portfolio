07 · ANATOMY

Anatomi Portfolio yang Profesional

01

Hero Section

Nama, headline, foto, 1 kalimat pitch. Recruiter mood-nya disetting di sini.

02

About Me

Story singkat: who you are, what you do, why it matters.

03

Skills & Tools

Stack technical + soft skills. Visual icons > paragraph.

04

Projects

3-5 project terbaik. Foto, deskripsi, link demo, link repo.

05

Experience

Magang, freelance, organisasi. Timeline format.

06

Contact

Email, LinkedIn, GitHub. CTA: "Let's collaborate".

08 · MASTER PROMPT

The Magic Prompt 🪄

Copy-paste ini ke AI coder kamu (Codex/Claude/Kiro), ubah bagian yang di-bold:

# PORTFOLIO PROMPT TEMPLATE

Buatkan website portfolio personal dengan info berikut:

Nama: [nama kamu]
Role: [Mahasiswa Teknik Informatika / Mobile Developer / dll]
Pitch 1 kalimat: [contoh: "Bikin web app yang bermanfaat dan keren"]
Skills: [HTML, CSS, JS, Python, Figma, dll]
Projects:
  1. [Nama project] - [deskripsi singkat] - [link]
  2. [Nama project] - [deskripsi singkat] - [link]
  3. [Nama project] - [deskripsi singkat] - [link]
Kontak:
  - Email: [email]
  - LinkedIn: [url]
  - GitHub: [username]

Style: Modern, dark theme, gradient accent, smooth animations.
Stack: HTML5 + CSS3 + Vanilla JS (no framework, biar gampang deploy).
Output: 3 file (index.html, style.css, script.js) — siap deploy ke Vercel.

Section yang harus ada: Hero, About, Skills, Projects, Experience, Contact.
Mobile responsive. Pakai Google Fonts. Tambahin smooth scroll animation.

💡 Lebih spesifik prompt → lebih bagus output. Sebut warna favorit, style referensi (Linear, Stripe, Vercel), atau vibe (formal, playful, minimalist).

09 · WORKFLOW

Workflow Lengkap dari Nol ke Live

1

Brainstorm

List project & skill yang mau ditampilkan. Siapin foto, link demo, link repo.

2

Prompt AI Coder

Pakai master prompt → AI generate index.html, style.css, script.js.

3

Save Files

Bikin folder portfolio/, save 3 file dari AI ke folder ini.

4

Preview Lokal

Double-click index.html → buka di browser. Cek dulu sebelum deploy.

5

Iterate

Nggak suka warnanya? Minta AI ubah. Mau tambah section? Minta AI tambahin.

6

Deploy ke Vercel

vercel --prod → live URL dapet dalam 30 detik.

7

Push GitHub

gh repo create → source code aman + bisa dipantau recruiter.

10 · CHEATSHEET

Commands yang Wajib Diingat

📦 Setup Project

mkdir portfolio
cd portfolio
git init -b main

🤖 Codex CLI

codex login --device-auth
codex "buatkan portfolio..."

🚀 Deploy Vercel

vercel login
vercel --prod --yes

🐙 Push GitHub

git add .
git commit -m "init portfolio"
gh repo create portfolio \
  --public --source=. \
  --remote=origin --push

🔄 Update & Redeploy

git add .
git commit -m "update"
git push
vercel --prod

✨ Custom Domain

vercel domains add \
  myname.vercel.app
11 · PRO TIPS

Tips Biar Portfolio Kamu Stand Out

01

Foto Project Real

Screenshot beneran > stock image. Bisa pakai screenshotone.com atau capture sendiri.

02

Tulis Case Study, Bukan Cuma "Saya Bikin X"

Format: Problem → Solution → Result. Recruiter suka yang bisa story-tell.

03

Custom Domain

namasaya.dev atau namasaya.com jauh lebih pro daripada random-string.vercel.app.

04

SEO Basic

Tambahin <meta description>, OG tags. Minta AI bikinin.

05

Dark Mode Toggle

Detail kecil yang bikin "wah, profesional banget" di mata recruiter.

06

Update Berkala

Project baru? Langsung tambahin. Portfolio yang stagnan = sinyal kamu nggak aktif.

12 · AVOID

Kesalahan yang Sering Terjadi ❌

🚫

Lorem Ipsum di Section "About"

AI kasih placeholder, kamu lupa ganti. Recruiter notice langsung skip.

🚫

Project Tanpa Link Demo

"I built X" tanpa link = klaim kosong. Selalu sertakan link live.

🚫

Foto Profil Selfie Toilet

Pakai foto formal atau semi-formal. Background bersih.

🚫

Bahasa Campur Aduk

Pilih satu: Indonesia atau English. Konsisten dari hero sampai footer.

🚫

Mobile Hancur

50%+ traffic mobile. Selalu test di HP sebelum nge-share.

🚫

Link Mati ke Project

Cek semua link sebelum kirim ke recruiter. 1 link mati = unprofessional.
