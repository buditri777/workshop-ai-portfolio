## 🎯 Untuk Pemateri

### Persiapan H-7
1. Baca `docs/00-runbook-pemateri.md` — paham timing & flow
2. Latihan demo dari nol → live URL (target < 10 menit) minimal **2 kali**
3. Siapkan akun demo terpisah (GitHub + Vercel)

### Persiapan H-1
1. Test slide deck di laptop yang akan dipakai
2. Test semua command di `docs/03-cheatsheet-command.md`
3. Bagikan handout `docs/01-panduan-persiapan.md` ke peserta (1 minggu sebelumnya)

### Hari-H
1. Buka slide deck (kontrol pakai panah ← →)
2. Ikuti runbook
3. Asisten standby pakai `docs/04-troubleshooting.md`

---

## 🎯 Untuk Peserta

### Sebelum Workshop
1. Baca `docs/01-panduan-persiapan.md`
2. Install semua tools (estimasi 60-90 menit)
3. Bawa laptop yang sudah ready ke workshop

### Saat Workshop
1. Print `docs/03-cheatsheet-command.md` — taruh di samping laptop
2. Ikuti pemateri
3. Manfaatkan `docs/02-prompt-templates.md` saat hands-on

### Setelah Workshop
1. Share URL portfolio di IG/LinkedIn
2. Update CV dengan link portfolio
3. Apply magang/kerja → portfolio jadi senjata utama
