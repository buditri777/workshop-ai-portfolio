## 🎯 Untuk Pemateri

### Persiapan H-7
1. Baca `docs/00-runbook-pemateri.md` — paham timing & flow online
2. **Setup Zoom Pro/Business** — generate meeting link + passcode
3. Latihan demo dari nol → live URL (target < 10 menit) minimal **2 kali**
4. Test screen share + recording (cloud + local)

### Persiapan H-1
1. Test slide deck di laptop yang akan dipakai
2. Test koneksi internet (min. 25 Mbps untuk pemateri)
3. Kirim link Zoom + passcode ke peserta via WA grup
4. Brief asisten/co-host dengan `docs/05-zoom-etiquette.md`

### Hari-H
1. Join Zoom **30 menit lebih awal** untuk setup
2. Test audio/video dengan asisten
3. Buka slide: `workshop-portofolio-ai.vercel.app`
4. Ikuti runbook online

---

## 🎯 Untuk Peserta

### Sebelum Workshop
1. Baca `docs/01-panduan-persiapan.md` — termasuk setup Zoom
2. **Install Zoom App** + headset/earphone ready
3. Install semua tools (estimasi 60-90 menit)
4. Test internet stabil (min. 10 Mbps)

### Saat Workshop
1. Join Zoom **15 menit lebih awal** (08:45 WIB)
2. Camera ON, mic mute saat tidak bicara
3. Pakai display name format: "Nama · Angkatan"
4. Manfaatkan `docs/02-prompt-templates.md` saat hands-on

### Setelah Workshop
1. Share URL portfolio di IG/LinkedIn dengan tag @fikomudb
2. Update CV dengan link portfolio
3. Akses recording (link dikirim H+1)
4. Apply magang/kerja → portfolio jadi senjata utama
