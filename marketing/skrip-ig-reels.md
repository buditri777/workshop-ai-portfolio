## 🎬 VIDEO #1 — "Pain Point" (H-14)

### Spec
- **Durasi:** 50 detik
- **Format:** 9:16 (1080x1920)
- **Music:** Upbeat tech / lo-fi beat
- **Style:** Fast cut, text overlay heavy

### Storyboard

| Scene | Durasi | Visual | Audio/SFX | Text Overlay |
|-------|--------|--------|-----------|--------------|
| **Hook** | 0-3s | Close-up tangan ngetik CV di Word, banyak font berantakan | Sound effect: glitch | **"Masih bikin CV pakai Word?"** |
| **Problem 1** | 3-8s | Cut: HRD scrolling cepet di laptop, mukanya bored | Soft "ugh" sound | "HRD baca CV cuma 6 detik..." |
| **Problem 2** | 8-13s | Tumpukan CV kertas, ada 1 yang dibuang | Paper crumple sound | "...trus berakhir di tempat sampah." |
| **Twist** | 13-18s | Zoom in HP, pakai 1 finger swipe IG, lihat link bio = portfolio website | Notification "ding" | **"Plot twist:"** |
| **Solution** | 18-25s | Quick cut 5x: laptop → AI prompt → code generate → URL muncul → handphone | Tech beat ramp up | "AI bisa bikinin portfolio kamu" |
| **Highlight** | 25-32s | Screen recording portfolio yang scroll halus, dark mode, animasi smooth | "Wow" subtle | "Profesional. Custom. Live URL." |
| **Proof** | 32-40s | Time-lapse: stopwatch nyala, dari 00:00 sampai 15:00 | Tick-tock cepet | **"Total: 15 menit. Tanpa coding."** |
| **CTA** | 40-50s | Static frame: poster workshop + QR code | Beat fade | **"Workshop FIKOM UDB. Daftar di link bio. Slot terbatas."** |

### Caption
```
Masih bikin CV pakai Word? 😬

Kabar baiknya: bikin portfolio profesional sekarang nggak butuh coding lagi. Cukup pakai AI, 15 menit kelar.
```

## 🎬 VIDEO #2 — "Speed Run Demo" (H-7)

### Spec
- **Durasi:** 60 detik
- **Format:** 9:16
- **Music:** Lo-fi tech vibe / EDM build-up
- **Style:** Screen recording dominan, narration voiceover

### Storyboard

| Scene | Durasi | Visual | Voiceover (VO) | Text Overlay |
|-------|--------|--------|----------------|--------------|
| **Hook** | 0-3s | Black screen, text muncul satu-satu | "Bikin portfolio profesional..." | **"From ZERO to LIVE..."** |
| **Setup** | 3-8s | Cursor klik buka ChatGPT/Claude di browser | "...dalam 60 detik. Ready?" | "Step 1: Buka AI Chat" |
| **Prompt** | 8-15s | Paste prompt panjang (di-fast-forward), tombol enter ditekan | "Paste prompt → enter" | **"Step 2: Master Prompt"** |
| **Generate** | 15-22s | AI ngetik output cepet (timelapse), code muncul layer demi layer | "AI ngegenerate semua..." | "Step 3: AI Generate Code" |
| **Save** | 22-28s | Cursor copy code → paste ke VS Code → save 3 file | "...kamu tinggal save." | "Step 4: Save 3 file" |
| **Preview** | 28-35s | Double-click `index.html` → browser kebuka → portfolio muncul keren banget | "Preview lokal." | **"Step 5: Preview"** |
| **Deploy** | 35-45s | Terminal: ketik `vercel --prod --yes` → progress bar → URL muncul | "Deploy 1 command." | "Step 6: vercel --prod" |
| **Live** | 45-55s | URL ke-copy → buka di tab baru → portfolio LIVE dengan domain `.vercel.app` | "Live di internet." | **"DONE. URL READY."** |
| **CTA** | 55-60s | Frame: poster workshop | "Mau belajar caranya?" | **"Workshop @fikomudb"** |

```
60 DETIK. ITU AJA YANG GUE BUTUH BUAT BIKIN PORTFOLIO. 🚀

⚡ Step 1: Buka AI Chat
📝 Step 2: Paste master prompt
✨ Step 3: AI generate code
💾 Step 4: Save file
👀 Step 5: Preview lokal
🚢 Step 6: Deploy
✅ DONE. URL siap di-share.

Caranya gw share di workshop FIKOM UDB minggu depan. GRATIS.

🎫 Slot 50 peserta — udah keisi 35.
📅 Sabtu, [tanggal]
🔗 Daftar di link bio

#fikomudb #portfoliodalam60detik #ai #vercel #portfolio
```

### Tips Produksi
- Pakai **OBS Studio** untuk screen recording (gratis)
- Speed up footage: 4x untuk bagian generate code, 2x untuk lainnya
- Pakai **ScreenStudio** atau **CleanShot X** untuk Mac (cursor highlights bagus)
- Transition pakai cut keras, jangan fade — biar punchy
- Tambah sound effect di tiap "step" (tactile clicks)

## 💡 Pro Tips

1. **Jangan lupa subtitle** — 80% Reels ditonton tanpa suara
2. **Hook 3 detik pertama** wajib kuat — itu yang nahan orang nggak swipe
3. **Pakai trending audio** kalau memungkinkan (cek IG trending audio dari akun yang spesialisasi)
4. **Tag dosen + akun resmi UDB** di setiap post
5. **CTA jelas di setiap frame:** "link bio", "daftar sekarang"
6. **Crosspost ke TikTok + Reels** sekaligus — beda audience
7. **Engage di komen 30 menit pertama** — algoritma IG sayang yang interaksi cepat
